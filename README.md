# Ocean

青山不改，绿水长流。

我主要使用 Python 和 Go，目前正在系统学习软件工程与后端开发，目标是成长为一名能够独立设计、开发和交付系统的开发工程师。

当前重点方向是 AI 应用、Agent 开发、微服务与分布式系统。我关注如何把 AI 能力落到真实业务场景中，也在持续补齐工程化、服务治理、部署运维和系统设计能力。

## Focus

- AI Applications: LLM 应用开发、RAG、工具调用、工作流自动化
- Agent Development: 任务规划、工具使用、记忆机制、多 Agent 协作
- Backend Engineering: Python、Go、API 服务、异步任务、数据处理
- Microservices: 服务拆分、服务通信、配置管理、可观测性、部署发布
- Distributed Systems: 可靠性、扩展性、一致性、任务调度与系统设计
